# RGB三原色分解体験サイト 🎨

高校生向けのインタラクティブな教育Webサイトで、JPEG画像のRGB三原色分解を視覚的に学習できます。

## 🎯 プロジェクト概要

このWebサイトは、デジタル画像処理の基礎概念であるRGB色彩システムを高校生が楽しく学べるよう設計された教育ツールです。画像をアップロードするだけで、リアルタイムでRGB各チャンネルに分解し、視覚的に理解を深めることができます。

### 主な学習目標
- デジタル画像の仕組み（ピクセル、RGB）の理解
- 人間の視覚システムとRGB色彩の関係
- 画像処理技術の実用的な応用例の紹介
- 科学技術分野における画像処理の重要性の理解

## 📚 データモデル・構造

### 画像処理データフロー
```javascript
File Input → Image Object → Canvas → ImageData → RGB分解 → 表示
```

### RGB分解アルゴリズム
```javascript
// 各ピクセルについて
for (let i = 0; i < imageData.length; i += 4) {
    const r = data[i];     // 赤チャンネル値
    const g = data[i + 1]; // 緑チャンネル値
    const b = data[i + 2]; // 青チャンネル値
    
    // グレースケール表示用に各チャンネルを分離
    redChannel[i] = redChannel[i + 1] = redChannel[i + 2] = r;
    greenChannel[i] = greenChannel[i + 1] = greenChannel[i + 2] = g;
    blueChannel[i] = blueChannel[i + 1] = blueChannel[i + 2] = b;
}
```

### ファイル制限
- **対応形式**: JPEG, PNG, GIF
- **最大サイズ**: 5MB
- **最大表示サイズ**: 600px（自動リサイズ）
